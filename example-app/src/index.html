Instant App Status

This example demonstrates how to check if your app is running as an instant app.

Checking instant app status...

Implementation

Here's how this example works:

1. Install the plugin

npm install @trentrand/capacitor-instant
# or
bun add @trentrand/capacitor-instant

2. Import and use the plugin

import { Instant } from '@trentrand/capacitor-instant';

async function checkInstantStatus() {
  try {
    const { value: isInstant } = await Instant.isInstantApp();
    console.log('Is instant app:', isInstant);

    if (isInstant) {
      // Running as Android Instant App or iOS App Clip
    } else {
      // Running as full app
    }
  } catch (error) {
    console.error('Failed to check instant app status:', error);
  }
}

What is an instant app?

Instant apps allow users to run your app without installation:

  • Android Instant Apps: Run directly from Google Play
  • iOS App Clips: Quick access to part of your app
  • Web: Always returns false (not applicable)
